Detailed Description

HiTechnic IR Link Sensor driver.

HTIRL-driver.h provides an API for the HiTechnic IR Link Sensor. Currently only the Power Functions API has been implemented.

Changelog:

  • 1.0: Initial release
  • 1.1: Minor changes
  • 1.2: Rewrite to make use of the new common.h API
  • 1.3: Clarified port numbering
  • 1.4: Removed inline functions

Credits:
