byte HTSMUXreadStatus ( tSensors  link  ) 

Read the status of the SMUX

The status byte is made up of the following bits:

| D7 | D6 | D4 | D3 | D2 | D1 | D1 | -D1 - HTSMUX_STAT_BATT: No battery voltage detected -D2 - HTSMUX_STAT_BUSY: Auto-dected in progress status -D3 - HTSMUX_STAT_HALT: Multiplexer is halted -D4 - HTSMUX_STAT_ERROR: Command error detected

Parameters:
link the SMUX port number
Returns:
the status byte
