-= Note =-
These drivers are for RobotC 2.00 and higher and are part of the
version 1.6 of the 3rd Party RobotC Drivers suite.


-= Changes in this version =-
* Fixed typos in HTGYRO-driver.h
* Newly added HiTechnic Magnetic Field Sensor + test programs
* Newly added Lego E-meter + test programs
* Newly added HiTechnic IR Link Driver NG - can now control motors using a simplified API.  See PFmotor() for more details
* Newly added HiTechnic Magnetic Field Sensor
* Newly added HiTechnic IR Link RCX control library
* MSDIST autoconfigures subsensor type
* NXTCAM allows Line Tracking mode

-= API Documentation =-
The complete API documentation can be found in the html folder.
Just point your browser at the index.html file.


-= Downloads and support =-
These drivers can also be downloaded from:
http://sourceforge.net/projects/rdpartyrobotcdr/

The documentation is hosted here:
http://rdpartyrobotcdr.sourceforge.net/documentation/index.html

For support questions, please use the RobotC 3rd party sensors forum:
http://www.robotc.net/forums/viewforum.php?f=41
